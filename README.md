🧊 Sobre o Projeto

O TEMPFY é um projeto de desenvolvimento web que simula uma empresa fictícia especializada em soluções de refrigeração para equipamentos eletrônicos.

A aplicação foi desenvolvida com foco em monitoramento e gerenciamento eficiente, proporcionando maior segurança e desempenho para dispositivos que necessitam de controle térmico.
