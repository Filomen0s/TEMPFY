How to test locally:
1. Copy this folder to your machine.
2. Place placeholder images in the /assets/ folder with the names listed above (capa-inicio.png, background-login-cadastro.png, tempfy-device.png).
3. Open `paginaInicial.html` in your browser (double-click or use a local server).

Notes:
- No backend required. All interactions are front-end only and simulated.
- Replace placeholder images in /assets/ with the Figma assets when ready.
